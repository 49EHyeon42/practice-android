# Android Example Application

이 프로젝트는 안드로이드 연습용 프로젝트 입니다.

## 회고

### Dagger Branch 회고

- 학습
    - list view 와 recycler view 사용
    - view, data binding 사용
    - 안드로이드 MVVM 패턴 사용
    - room 라이브러리 사용
    - dagger 라이브러리 사용
    - 안드로이드 패키지 구조 고민
- 부족한 점, 이후 학습할 것
    - RxJava 를 활용한 안드로이드 비동기 처리
    - retrofit2 라이브러리를 활용한 통신
    - hilt 라이브러리
